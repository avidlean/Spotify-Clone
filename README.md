# Spotify Clone

## Overview
Spotify Clone is a web-based music player built with HTML, CSS, and JavaScript. It replicates core Spotify features such as playing, pausing, and navigating through a playlist of songs with song metadata and cover images.

## Features
- Play, pause, and navigate songs using a master control or individual song selections.
- Dynamic update of song details and cover images as different songs are played.
- Interactive progress bar that shows current playback and allows seeking.
- Responsive design ensuring good user experience across devices.
- Custom styling using Google Fonts (Ubuntu and Varela Round).

## Technologies Used
- HTML for structure
- CSS for styling and layout
- JavaScript for audio playback logic and UI updates

## How to Use
1. Clone or download the repository.
2. Open `index.html` in a modern web browser.
3. Use the play/pause button, individual song play buttons, or next/previous controls to listen to the playlist.

## Customization
To add more songs, update the `songs` array in `script.js` with the song name, file path, and cover image path.
